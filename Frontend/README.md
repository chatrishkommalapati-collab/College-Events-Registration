﻿# College Events Registration

A React + Vite web app for managing student registrations and college event dashboards.

## Project Description

This project demonstrates React fundamentals using a single-page application built with Vite. It includes student registration, login flow, dashboard navigation, protected routes, conditional UI states, and dynamic data rendering.

## Features Implemented

- Home page with reusable dashboard cards
- Student registration form with validation and controlled inputs
- Login page with password visibility toggle and loading state
- Dashboard layout with sidebar navigation and nested routes
- Search/filter for registered students
- Dynamic cards and table rendering using `map()`
- 404 Not Found page
- Responsive layout and CSS modules

## React Concepts Used

- Functional components
- Props and reusable components
- `useState` for state management
- Controlled components and two-way data binding
- Event handling with `onClick`, `onChange`, and `onSubmit`
- Conditional rendering using ternary operators and logical `&&`
- React Router DOM: `BrowserRouter`, `Routes`, `Route`, `Link`, `NavLink`, `useNavigate`, `useParams`
- Module exports and project structure
- Styling with CSS modules and a global stylesheet

## Class Components vs Functional Components

Class components use ES6 classes and lifecycle methods, while functional components are plain functions with hooks. This project uses functional components for simpler, modern React code.

## Props and `main.jsx`

Props are read-only values passed from parent components to children. They make components reusable and configurable. `main.jsx` initializes React and wraps the app with `BrowserRouter` so routing works across pages.

## Styling Approach

- Global base styles in `src/index.css`
- Component-specific styling with CSS modules
- Reusable class names and responsive layouts

## Run the Project

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
